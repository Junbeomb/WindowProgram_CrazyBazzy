# 제목 : CrazyBazzy  
장르 : RPG  
개요 : 윈도우 프로그램 기말 프로젝트  
유사 게임 : 크레이지 아케이드 (보스 모드)  
참여 인원 : 박준범, 박강수, 최재영  

## 역할 분담  
*박준범  
  1. 보스 구현, 보스 애니메이션,  
  2. 캐릭터 이동, 공격, 스킬  
  3. UI
